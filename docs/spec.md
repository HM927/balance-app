# 残高管理アプリ 仕様書

## 第1段階：基準情報の登録

### 作る機能
- 現在の残高を入力できる
- 基準日を入力できる
- 入力内容を保存できる
- 保存した内容を画面に表示できる

### 画面
- 残高入力欄
- 基準日入力欄
- 保存ボタン
- 保存済み残高の表示
- 保存済み基準日の表示

### データ
- balance
- baseDate

## 第2段階：収支の登録

### 作る機能
- 収入・支出を入力できる
- 日付を入力できる
- 金額を入力できる
- 内容メモを入力できる
- 入力した収支を保存できる
- 保存した収支を一覧表示できる

### 画面
- 種別選択欄
- 日付入力欄
- 金額入力欄
- メモ入力欄
- 追加ボタン
- 収支一覧

### データ
- id
- type
- date
- amount
- memo
